Politique de confidentialité

Dernière mise à jour : mars 2025

Aperçu

Venuly est conçu dans le respect de votre vie privée. Nous ne créons pas de comptes utilisateurs, ne collectons pas d'informations personnelles et ne suivons pas votre activité. Cette politique explique comment l'application gère les données.

Données que nous collectons

Venuly stocke les préférences suivantes localement sur votre appareil :

  • Sélection de ville ou de code postal — utilisée pour vous montrer les événements près de chez vous
  • Événements sauvegardés — stockés localement sur votre appareil uniquement
  • Préférences de catégories — stockées localement sur votre appareil uniquement

Toutes ces données sont stockées via le système UserDefaults d'Apple et ne quittent jamais votre appareil.

Données que nous ne collectons pas

  • Aucun compte utilisateur ni inscription
  • Aucune information personnelle (nom, courriel, numéro de téléphone)
  • Aucun suivi d'utilisation ni analytique
  • Aucun identifiant publicitaire
  • Aucun cookie ni suivi inter-applications

Données de localisation

Venuly peut demander l'accès à la localisation de votre appareil pour vous aider à trouver des événements à proximité. Les données de localisation sont :

  • Accédées uniquement lorsque vous accordez explicitement la permission
  • Utilisées uniquement pour déterminer la ville la plus proche pour les listes d'événements
  • Jamais stockées sur un serveur
  • Jamais partagées avec des tiers

Services infonuagiques

Venuly utilise Google Firestore pour stocker les données d'événements. Cette connexion est en lecture seule — aucune donnée utilisateur n'est jamais écrite dans la base de données. L'application ne fait que récupérer les informations sur les événements ; elle n'envoie aucune information vous concernant.

Données de tiers

Les informations sur les événements affichées dans Venuly proviennent de l'API Ticketmaster. Ces données sont affichées selon les conditions d'utilisation de Ticketmaster. Venuly ne partage aucune donnée utilisateur avec Ticketmaster ni aucun autre tiers.

Conservation des données

Puisque toutes les préférences utilisateur sont stockées localement sur votre appareil via UserDefaults, vous avez un contrôle total sur vos données. La désinstallation de l'application supprime toutes les préférences stockées.

Vie privée des enfants

Venuly ne collecte sciemment aucune information personnelle auprès des enfants. Puisque l'application ne collecte aucune information personnelle d'aucun utilisateur, elle est sécuritaire pour tous les âges.

Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité de temps à autre. Tout changement sera reflété sur cette page avec une date de mise à jour.
